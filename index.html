 會員資格認定標準：
以同一要保人投保新光人壽保單，在同一通路(說明一)之近一年實繳保費換算(說明二)累積達500萬(含)以上者，即為2019年新光人壽VIP極享俱樂部會員，會員資格每半年(6月底)將檢核更新。
 說明一、通路說明 :
同一要保人計算保費，依各通路投保之保單實繳保費換算分別累積計算，說明如下 :
1. 業務員通路 : 經由本公司所轄業務員投保。
2. 保經代通路 : 經由與本公司合作之各 銀行/證券／保代公司等通路向本公司投保，各銀行/證券／保代公司之保費分別累積計算。

 說明二 : 保費換算計算標準如下 :
1. 保單僅以有效保單計算，並排除已繳費期滿、減額繳清、展期定期、停效之保單。且不包含意外險、團體險及旅平險之保費。
(1). 躉繳保單以實繳保費除以20換算
(2). 期繳為2至5年期之保單以實繳保費除以2換算
(3). 期繳為其他年期之保單以實繳保費換算
(4). 外幣保單以前述(1)~(3)點之保費乘上1.5倍換算
(5). 泛保障型(保障型與健康型)以前述(1)~(4)點之保費乘上1.5倍換算
(6). 投資型保單僅計算定期繳納的實繳保費，不含另行約定增額保費與單筆投入。
2. 超級新光保戶加分計算條件
(1). 前項各款保費以新光銀行(信用卡或轉帳扣繳)繳交保費金額乘上1.2倍換算
(2). 近一年新壽保單貸款已繳利息，其金額以1倍併入第1項換算後保費合併計算
 專屬服務說明

為使新光極享俱樂部的會員服務更便捷化與行動化，特製作專屬Merlin APP，提供給極享俱樂部會員下載使用。會員可透過Merlin APP享受到的服務如下 :
1. 極享旅遊 : 為新光貴賓特別規劃，專屬客製旅遊行程或體驗，獨特並稀有，相關活動提供給極享俱樂部會員及其親友可報名參加，極享旅遊分為三大主軸 :
(1). 專屬旅程 : 每季推出季節性與獨特的旅程，讓會員可隨四季享受世界旅遊特色，悠遊於世界各種頂級獨特之行程。
(2). 特殊體驗 : 在每次的旅遊，增加短暫浸濡在地文化民情，讓自己的旅程以更深度、更彈性的方式，享受當次的假期。
(3). 其他服務 : 不定期舉辦留學教育說明會、世界知名酒莊品酒活動、參與特殊活動、展覽等活動。
(4). 會員可於專屬Merlin APP上進行快速瀏覽所有極享旅遊行程，欲詢問或報名極享旅遊，可透過Merlin APP上的一鍵撥打，直接與禮賓服務公司Quintessentially聯絡接洽，自費參與。
2. 酒店預訂 : 提供世界頂級酒店的預訂服務 (酒店列表請見專屬Merlin APP)，且透過禮賓服務公司Quintessentially預訂，享有每日免費早餐，另有客房升級、提早入住、延遲退房的專屬禮遇(視飯店供應情況而定)。欲詢問或報名酒店預訂，會員可透過梅林APP上的一鍵撥打，直接與禮賓服務公司Quintessentially聯絡接洽，自費參與。
3. 專線服務 : 透過Merlin APP即可一鍵撥打(免撥號) 至專線服務，專線依諮詢功能分為兩種 :
(1). 極享服務諮詢 : 關於Merlin APP上的極享旅遊與酒店預訂等相關諮詢與預訂需求，可點選專線服務之極享旅遊諮詢，將有禮賓服務公司Quintessentially專人為您服務。
(2). (2) 保單服務諮詢 : 保單相關資訊、服務及新光人壽相關活動資訊洽詢，點選專線服務之保單服務諮詢，將有新光客服專人為您服務。
(3).
